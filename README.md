# dotted.me
Convertir imagen a carácteres Braille

```
dotted.me 0.1
Santiago C. <santiagocezar2013@gmail.com>
Convertir imagen a carácteres Braille

USAGE:
    braille_gfx [FLAGS] [OPTIONS] <image>

ARGS:
    <image>    Archivo de imagen

FLAGS:
    -h, --help       Prints help information
    -i, --invert     Invertir para usar en temas oscuros
    -V, --version    Prints version information

OPTIONS:
    -l, --level <level>    Nivel de luminosidad para el umbral (0-255) [default: 128]
    -w, --width <width>    Cantidad de puntos horizontales (1 caracter = 2 puntos) [default: 32]
```
